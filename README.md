# hellcord-private

Простенький бот для приваток на сервере [HellCord](https://discord.gg/hellcord)

### Установка и запуск
```bash
make dev
```

### Сборка

```bash
make build
```

This project was created using `bun init` in bun v1.1.30. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
